cycle(fun,cycleTime,ttl[,param1,param2....paramn]) 方法

描述

延迟执行,执行delay方法,会返回一个对象,对象有stop()方法停止执行
fun:要执行的函数体 cycleTime:循环周期:延迟多久后执行,单位毫秒 ttl:最大的执行时间,超过此时间,停止循环执行

例子

(function(Q){
	var thread=Q.cycle(function(){
		var args=arguments;
		console.log("param1:"+args[0]);//>>>:11
		console.log("param1:"+args[1]);//>>>:12
		console.log("param1:"+args[2]);//>>>:aa
	},1000,10000,11,12,"aa");//第1000毫秒执行一次,最长执行10000毫秒

})(Qmik)
