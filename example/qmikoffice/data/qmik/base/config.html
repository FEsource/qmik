config(opts[,config]) 方法

描述

配置Qmik参数,或生成一份新的参数
1.当不传参数时,返回qmik的参数配置信息
1.当只传输一个参数时,默认对qmik做参数设置
2.当传输两个参数时,是对第二个参数config做参数配置
几个重要参数: context : "/qmik/qmikoffice/" //上下文地址,这个地址的设置,这个的配置,会影响到 url(url) 方法的返回值 debug : true //调式模式

例子

(function(Q){
	Q.config({
		context : "/qmik/qmikoffice/" //,
		debug : true
	});

})(Qmik)
