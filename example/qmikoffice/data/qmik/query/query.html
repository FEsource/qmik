Q("#id")

描述

根据id查找,如果有多个重复的id,只取第一个,返回一个类数组对象

例子

<div id="bak">
 <div id="ban">
	
 </div>	
</div>
(function(Q){
 var qs=Q("#bak"); 
})(Qmik)

Q(".class")

描述

根据class查找,返回一个类数组对象

例子

<div id="bak" class="ab">
	<div id="ban">
	
	</div>
</div>
(function(Q){
	var qs=Q(".ab"); 
})(Qmik)

Q("query1 query2")

描述

根据多个条件来查找,query1是query2的父级以上的节点(父,祖父,祖祖父)

例子

<div id="bak">
	<div id="ban" class="cb">
		<div class="cb">
	
		</div>
	</div>
	<div class="cb">
	
	</div>	
</div>
(function(Q){
	var qs=Q("#bak .cb"); 
})(Qmik)

		</div>
	</div>
