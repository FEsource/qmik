Qmik.inherit()函数用于类继承。

Qmik.inherit(subClass, superClass);

参数 描述
subClass 子类
superClass 父类 

  	function Person(){

    }
    $.extend(Person.prototype, {
      show: function(msg){
        $.log("show ", msg);
      }
    });

    function Boy(){};

    $.inherit(Boy, Person);

    var boy = new Boy();
    boy.show("user");
